§01 Overview

A small vessel that does one thing well.

jltool wraps the official jarvislabs Python SDK behind a Typer command-line interface and ships as a single console script. It manages GPU instances, startup scripts, persistent filesystems, SSH keys, and account information — every operation a single command, every command a thin, predictable surface over the underlying API.

The entire CLI lives in a single file: jltool/cli.py. There is no global state, no configuration class, no plugin system. Each subcommand opens a client, makes one call, prints a Rich-formatted table, and exits.

§02 Configure

One env file. Sensible defaults.

Place a .env in the project root or any parent directory. JL_API_KEY is the only required value — the rest are optional defaults applied when the matching flag is omitted on jltool instances create.

# Required — get yours at jarvislabs.ai/settings/api-keys
JL_API_KEY=jl_xxxxxxxxxxxxxxxxxxxxxxxxx

# Defaults for `jltool instances create`
JL_DEFAULT_GPU_TYPE=A100
JL_DEFAULT_NUM_GPUS=1
JL_DEFAULT_TEMPLATE=pytorch
JL_DEFAULT_STORAGE=40
JL_DEFAULT_HTTP_PORTS=7860,8080

# Optional — defaults for attached resources
JL_DEFAULT_SCRIPT_ID=
JL_DEFAULT_FS_ID=
§03 Commands

Five groups. Twenty‑something verbs.

Run jltool --help for the full reference. Each group is a typer.Typer sub-app mounted under the root command.

jltool instances

Compute

  • list · get · create
  • pause · resume · rename
  • destroy · pause-all
jltool fs

Filesystems

  • list · create
  • edit · remove
jltool scripts

Startup scripts

  • list · add
  • update · remove
jltool keys

SSH keys

  • list · add · remove
jltool account

Account

  • balance · metrics
  • gpus · templates
shortcuts

Top‑level

  • jltool ls   → instances list
  • jltool balance → account balance

§05 Exit codes

Errors map to exit codes.

Every SDK exception is translated to a distinct exit code so commands compose cleanly in shell scripts and CI pipelines.

CodeClassMeaning
0—Success.
1JarvislabsErrorGeneric SDK failure that did not match a more specific class.
2AuthErrorMissing or invalid token. Set JL_API_KEY in your environment or .env.
3NotFoundErrorThe requested resource (instance, script, fs, key) does not exist.
4ValidationErrorInvalid input — usually a missing or out-of-range flag.
5APIErrorUpstream API returned a non-2xx status. The HTTP code is printed to stderr.
130—Interrupted by Ctrl-C.
